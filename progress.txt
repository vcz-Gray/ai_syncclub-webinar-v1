## Codebase Patterns
- Static-first is the right shape for this repo because the remote repository is empty and the deliverable is mostly HTML and markdown.
- The webinar deck already exists as a self-contained HTML file, so the site should link to it rather than trying to rebuild it with a framework.
- Participant resources should exist both as browsable pages and raw markdown downloads.
- Skills need concise `SKILL.md` files plus `agents/openai.yaml` metadata; the site should explain how to use them without requiring Codex expertise.
