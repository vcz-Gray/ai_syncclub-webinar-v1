# Event Promo Copy

Use this skill when you need short event copy.

## Output Types

- 카톡/슬랙 공지
- 커뮤니티 홍보 글
- 후속 다시보기 안내
- 책/저자 중심 소개문

## Rules

- Open with one strong promise.
- Keep channel-specific versions short.
- Do not oversell vague transformation.
- End with one clear action: 보기, 참여, 다운로드, 공유.

Read `references/examples.md` for channel patterns.
