# Webinar Follow-up Kit

Use this skill to turn a webinar into useful aftercare content.

## Deliverables

- 참가자 치트시트
- 자주 묻는 질문
- 1일/3일/7일 액션 가이드
- 요약 공지문

## Workflow

1. Identify the main promise of the webinar.
2. Distill only the actions participants can take immediately.
3. Group friction points into FAQ.
4. Keep docs short and reusable.

## Rules

- Prefer checklists over essays.
- Write for tired participants reviewing notes after the event.
- Always include a “next action” section.
