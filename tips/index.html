이번 이벤트 참여자를 위한 꿀팁 모음

웨비나를 보고 끝내지 않도록, 오늘 바로 써먹는 순서와 자료만 골라 정리했습니다. 목표는 “아이디어를 PRD로 바꾸고, Stitch로 화면을 만들고, 다음 액션까지 이어가기”입니다.

오늘 바로 해야 할 5단계

  • 웨비나 슬라이드를 다시 보며 내 서비스 유형을 하나 고릅니다.
  • PRD 템플릿 4문항에 답합니다.
  • Stitch 프롬프트 팩에서 가장 비슷한 예시를 복사합니다.
  • 첫 화면 1개, 추가 화면 2개 정도만 먼저 만듭니다.
  • 다음 3일 액션 가이드대로 검토와 수정 루프를 돌립니다.

많이 막히는 지점

  • 아이디어가 너무 크다 → 핵심 사용자 1명, 핵심 문제 1개로 축소
  • 기능이 너무 많다 → 반드시 필요한 3개만 남기기
  • 화면이 안 나온다 → 페이지 수를 먼저 지정하기
  • 결과가 엉뚱하다 → 수정 프롬프트에서 우선순위를 다시 명시
  • 다음 단계가 막막하다 → 후속 액션 가이드를 그대로 따라가기

PRD 템플릿

웨비나 다음 날부터 일주일 안에 실제 결과물을 만드는 루틴입니다.
